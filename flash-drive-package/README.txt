======================================== 
  Gas Cylinder Dashboard - Installation 
======================================== 
 
REQUIREMENTS: 
- Windows 10/11 
- Node.js 18+ installed 
- Internet connection 
 
INSTALLATION: 
1. Copy this entire folder to each computer 
2. Run the appropriate installer: 
   - INSTALL-OPERATOR.bat  (for data entry staff) 
   - INSTALL-MANAGER.bat   (for supervisors) 
   - INSTALL-ADMIN.bat     (for administrators) 
 
FEATURES: 
- Auto-login based on terminal type 
- Real-time data sync across all devices 
- Works with web version at: https://max-gas-transactions.vercel.app 
- Offline support with auto-sync when online 
 
SUPPORT: 
Backend: https://max-gas-backend.onrender.com 
Web App: https://max-gas-transactions.vercel.app
